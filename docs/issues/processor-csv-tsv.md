# CSV/TSV サポート

## 概要

表形式データ（`.csv`, `.tsv`）をインデックス対象に追加する。

## 実装方針

### テーブルモード（デフォルト）

ヘッダー＋行を「カラム名: 値」形式に自然言語化してエンベディング。

例:
```
名前: 田中太郎, 年齢: 30, 部署: 開発
名前: 佐藤花子, 年齢: 25, 部署: デザイン
```

- 1 行 = 1 チャンク（行数が多い場合はバッチ化も検討）
- `packages/cli/src/cli/processor/` に `csv_processor.py` を追加

### `--as-text` オプション（将来）

文字ベースでそのまま取り込む（Excel 方眼紙的運用向け）。
優先度低。テーブルモードの実装後に検討。

## 落としたもの

- EPUB: ニッチ。ビューア管理が主流でローカルファイルとして持つユーザーが少ない
- RST: Markdown への移行が進んでいる。新規プロジェクトでの採用が減少傾向
