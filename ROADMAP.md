# ROADMAP

## コンセプト

**「AI が使いやすい日本語ローカル検索エンジン」**

差別化の軸：日本語精度 × AI可読性 × 軽量・ローカル

---

## Milestone 1: AI可読性の改善 ✅ほぼ完了

**ゴール**: 検索結果が AI にとって精度よく・適切な量で渡される状態

### タスク
- [x] ランキングの改善（RRF: Reciprocal Rank Fusion の導入）
- [x] スキーマ正規化（doc_meta + doc_chunk の 2 テーブル構成）
- [x] Adjacent chunks によるコンテキスト拡張（ヒットチャンクの前後を結合して返す）
- [x] チャンク長の見直し（chunk_size=1000 / overlap=150 / min_chunk=200。文境界優先分割・見出しプレフィックス付与を導入）
- [ ] FTS 日本語対応の改善（現状: `simple` トークナイザーが日本語連続文字列を分割できず長文でヒットしにくい。形態素解析による事前分割などを検討）
